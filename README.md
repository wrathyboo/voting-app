## Description

A voting chess tournament website

## Installation

```bash
$ npm install
```

## Running the app

API
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
Frontend
```bash
# development
$ ng serve -o

```
